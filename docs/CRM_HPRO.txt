
Documentação da API - WebService HPro
1. Objetivo
Este documento descreve os endpoints disponíveis no WebService da HPro, utilizado para integração do sistema ERP da IS. A
comunicação é feita via requisições HTTP utilizando o formato JSON.
2. Formato dos dados
• Entrada e saída: JSON
• Codificação: UTF-8
3. Dados Gerais
URL BASE: http://hda0897nxnd.sn.mynetname.net:9080
Usuário: IAAGENDA
Senha: 1AG3NDA
4. Token
Todas as requisições exigem o envio de um Token válido no corpo da requisição.
Exemplo de chamada: http://localhost:82/api-auth-signin?Username=XXXX&Password=XXXX
Método HTTP: GET
Parâmetros:
• Username (texto, obrigatório)
• Password (texto, obrigatório)
Retorno (JSON):
{
"Token":"123456"
}

◼ Inclusão/alteração de atividades (AGENDA)
Descrição: Insere ou atualiza atividades.
Exemplo de chamada: http://localhost:82/api-services-atividadesajustes?Token=123456&Empresa=2&ChavePrimaria=5;17
Método HTTP: POST
Parâmetros obrigatórios:
• Token (texto, obrigatório)
• Empresa (inteiro, obrigatório)
• Celula (inteiro, obrigatório)
• Cliente (inteiro, obrigatório)
• Vendedor (inteiro, obrigatório)
• ChavePrimaria (texto, opcional, 999999;99999)
Envio (JSON):
[{
"Data": texto, aaaa-mm-dd, opcional,
"Hora": texto, hhmm, opcional,
"DataProximaAcao": texto, aaaa-mm-dd, opcional,
"HoraProximaAcao": texto, hhmm, opcional,
"HistoricoAtivadade": texto, opcional,
"TipoAtividade": inteiro, opcional,
"PlanoAcao": texto, opcional,
}]
Retorno:
Atividade/agenda inclusa/alterada com sucesso!
6. Erros de validações
Possíveis erros nas chamadas.
• "Token inválido!"
• "O parâmetro "xxxxx" é obrigatório!"
• Atividade não encontrada!
• Operação inválida para a realização da atividade! A atividade não está "Pendente"!
• Operação inválida para a realização da atividade! Não é possível baixar a atividade com data futura!
• Operação inválida para a realização da atividade! Não é permitido incluir visitas aos Sábados e Domingos!
• Operação inválida para a realização da atividade! Horário inválido! Horário permitido para visitas das 08:00h às 18:00h
• Operação inválida para a realização da atividade! Não existe vínculo entre vendedor, cliente e célula!
